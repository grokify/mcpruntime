## [0.1.0] - 2026-01-10

### Added

- Runtime type wrapping mcp.Server with dual execution modes (library and server)
- New() constructor with Options for runtime configuration
- MCPServer() escape hatch for advanced MCP SDK access
- Implementation() getter for server identity
- AddTool[In, Out]() generic helper with automatic schema inference
- AddToolHandler() low-level tool registration
- CallTool() for library-mode direct invocation
- RemoveTools(), ListTools(), HasTool(), ToolCount() tool management methods
- AddPrompt() for prompt registration
- GetPrompt() for library-mode direct invocation
- RemovePrompts(), ListPrompts(), HasPrompt(), PromptCount() prompt management methods
- AddResource() for static resource registration
- AddResourceTemplate() for dynamic URI template resources (server mode only)
- ReadResource() for library-mode direct invocation
- RemoveResources(), RemoveResourceTemplates() resource removal methods
- ListResources(), HasResource(), ResourceCount() resource management methods
- ServeStdio() for stdio transport (subprocess mode)
- ServeIO() for custom IO streams
- Serve() for custom MCP transports
- Connect() for session-based connections
- StreamableHTTPHandler() for HTTP/SSE transport
- SSEHandler() for legacy SSE transport
- InMemorySession() for testing with full MCP semantics

### Documentation

- Comprehensive README with library and server mode examples
- Feature comparison matrix (library vs server mode)
- MCP feature adoption guidance
- Full API reference documentation
